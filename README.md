# PROJETO CALCULADORA DE IMC

## DESCRIÇÃO
ESTE É O MEU PRIMEIRO EXERCÍCIO PRÁTICO NO CURSO DE ENGENHARIA DA COMPUTAÇÃO. O PROJETO CONSISTE EM UMA INTERFACE WEB SIMPLES QUE CALCULA O ÍNDICE DE MASSA CORPORAL (IMC) DE UM USUÁRIO COM BASE NO PESO E NA ALTURA FORNECIDOS.

## OBJETIVO
O OBJETIVO DESTE PROJETO É APLICAR CONHECIMENTOS INICIAIS DE DESENVOLVIMENTO WEB, FOCO EM LÓGICA DE PROGRAMAÇÃO COM JAVASCRIPT E ESTRUTURAÇÃO DE ELEMENTOS COM HTML.

## FUNCIONALIDADES
- ENTRADA DE DADOS (PESO E ALTURA).
- PROCESSAMENTO DO CÁLCULO DE IMC VIA JAVASCRIPT.
- EXIBIÇÃO DO RESULTADO E DA CATEGORIA DE PESO (EX: ABAIXO DO PESO, PESO IDEAL, ETC).

## TECNOLOGIAS UTILIZADAS
- HTML5
- JAVASCRIPT (VANILLA)

## COMO EXECUTAR
1. FAÇA O DOWNLOAD OU CLONE DO REPOSITÓRIO.
2. ABRA O ARQUIVO INDEX.HTML EM QUALQUER NAVEGADOR WEB.
